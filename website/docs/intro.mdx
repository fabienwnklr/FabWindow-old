# Fab Window Intro

Let's discover **Fabwindow in less than 5 minutes**.

## Getting Started only with Fab Window

Get started by **import Fabw Window or FabWindowManager script**.

<Tabs>
<TabItem value="npm" label="npm">

```bash
npm i fabwindow
```

</TabItem>
<TabItem value="yarn" label="yarn">

```bash
yarn add fabwindow
```

</TabItem>
</Tabs>

## Getting Started with Fab Window Manager

<Tabs>
<TabItem value="npm" label="npm">

```bash
npm i fabwindowmanager
```

</TabItem>
<TabItem value="yarn" label="yarn">

```bash
yarn add fabwindowmanager
```

</TabItem>
</Tabs>
