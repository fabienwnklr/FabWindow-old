# Getting started with FabWindow

Let's discover **FabWindow in less than 5 minutes**.
FabWindow is a JavaScript library that provides a rich user experience for display Window Modal. It supports single and multi-windows, easy customization, infinite scrolling and more.

## Installation

FabWindow has <b>NO</b> dependencies. We include styles and provide SCSS files for you to customize with any framework you like.

<Tabs>
<TabItem value="npm" label="npm">

```bash
npm i fabwindow
```

</TabItem>
<TabItem value="yarn" label="yarn">

```bash
yarn add fabwindow
```

</TabItem>
</Tabs>

## Install with CDN

The fastest way to add FabWindow to your project is to include the js and css from cdnjs.

```html
<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/" />
<script src="https://cdnjs.cloudflare.com/ajax/libs/"></script>
```
